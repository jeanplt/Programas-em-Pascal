# Programas-em-Pascal

Este repositório possui cerca de 100 programas em Pascal com finalidades diversas para aprender a programar. Os programas existentes resolvem desde problemas mais triviais como operações matemáticas, até cálculos em matriz. A ordem de complexidade dos programas aumenta com base nas listas, sendo as primeiras listas mais simples. Abaixo temos a organização das listas com os enunciados/problemas que cada um dos programas solucionam. Devido a quantidade de exercícios, recomendo utilizar CTRL+F para localizar os exercícios com maior facilidade.
